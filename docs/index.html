How it works

Every day at 08:00 CEST, GitHub Actions fetches:

  • Upcoming matches and bookmaker odds (the-odds-api.com)
  • Team Elo ratings (Clubelo.com)
  • Form and head-to-head (football-data.co.uk)
  • AI commentary (Anthropic Claude with web_search)

Results are exported to data/top10.json served by GitHub Pages. The mobile app downloads this JSON - no traditional backend required.

API endpoints (static JSON)

  • GET data/top10.json - lista top 10 meczow dnia
  • GET data/matches/{id}.json - pelne insights dla 1 meczu
