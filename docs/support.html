Najczesciej zadawane pytania (FAQ)

Skad pochodza dane o meczach?

Kursy bukmacherskie pobieramy raz dziennie z publicznego API the-odds-api.com (agreguje Betclic, Pinnacle, Bet365 i innych). Forma druzyn z football-data.co.uk i clubelo.com. Komentarz analityczny generuje model AI Anthropic Claude.

Jak czesto dane sie odswiezaja?

Codziennie o 08:00 czasu polskiego (06:00 UTC). GitHub Actions cron uruchamia skrypt, ktory generuje nowy plik top10.json i automatycznie publikuje go na GitHub Pages. Aplikacja pobiera nowe dane przy nastepnym otwarciu.

Czy moge zaobstawiac przez aplikacje?

Nie. ProbWin AI to aplikacja informacyjna - prezentuje statystyki i kursy, ale nie umozliwia obstawiania. Zaden link nie prowadzi do bukmachera. Nie wspolpracujemy z zadnym operatorem hazardowym.

Czy potrzebuje konta? Czy zbieracie moje dane?

Nie. Aplikacja nie wymaga rejestracji ani logowania. Nie zbieramy emaila, lokalizacji, identyfikatorow reklamowych ani historii uzycia. Wszystkie ustawienia (np. wybor ulubionej druzyny) sa zapisywane wylacznie lokalnie na Twoim urzadzeniu. Pelne info: polityka prywatnosci.

Co oznacza "prawdopodobienstwo 64%"?

To implikowane prawdopodobienstwo wyliczone z kursow bukmacherskich. Wzor: P = (1 / kurs) / suma odwroconych kursow. Przyklad: kurs 1.85 -> P ~ 51% po normalizacji marży bukmachera. Liczba pokazuje na ile rynek "wycenia" dany wynik - nie jest to gwarancja zwyciestwa.

Skad sie biora 'mock' dane?

Jezeli widzisz w aplikacji adnotacje "Betclic (mock)" lub "AI commentary disabled", oznacza to ze cron nie mial dostepu do API danego dnia (np. limit darmowego planu the-odds-api wyczerpany). Dane pokazuja realistyczne przyklady, ale nie sa kursami bieżącymi. Podejmuje dzialania zeby ten stan minimalizowac.

Aplikacja sie zawiesza / nie ladnja sie dane.

1) Sprawdz polaczenie internetowe. 2) Zamknij aplikacje calkowicie i otworz ponownie. 3) Jezeli problem trwa - napisz na contact@probwinai.app z opisem (model telefonu, wersja iOS/Android, screenshot bledu).

Czy aplikacja jest darmowa? Czy beda reklamy?

Tak, aplikacja jest w 100% darmowa. Nie ma reklam. Nie ma platnych funkcji premium (na ten moment). Utrzymanie pokrywa GitHub Pages (darmowy hosting) + GitHub Actions (darmowe minuty na cron).

Mam pomysl na nowa funkcje. Gdzie zglosic?

Najlepiej przez GitHub Issues (publiczne, transparentne) lub mailem na contact@probwinai.app. Czytam kazda wiadomosc.

Jak usunac aplikacje i wszystkie moje dane?

Wszystkie dane sa lokalne na urzadzeniu. Wystarczy odinstalowac aplikacje (przytrzymaj ikone -> Usun aplikacje). Nic nie pozostaje na zadnym serwerze, bo nic na serwer nigdy nie wyslalismy.

Hazard - odpowiedzialna gra. Hazard moze uzalezniac. Pomoc 24/7 (Polska): 801 199 990 (bezplatna). Miedzynarodowo: www.begambleaware.org. Aplikacja jest dla osob 18+.

Zglaszanie bledow

Jezeli zauwazyles blad w danych (zly wynik, brakujacy mecz, dziwny komentarz AI) - prosze daj znac. Zaznacz w mailu: nazwy druzyn, date meczu oraz screenshot. Naprawie i zaktualizuje dane przy nastepnym refreshu.

Status systemu

Status danych mozesz sprawdzic bezposrednio: top10.json. Pole generated_at pokazuje, kiedy ostatnio dane byly odswiezone. Jezeli starsze niz 36h - cron sie zepsul, zglos prosze.
