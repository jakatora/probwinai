## 2. Scope of Data Collected

ProbWin AI COLLECTS only:

- **Anonymous app settings** (e.g., API server address) stored locally in device memory (SharedPreferences). NOT sent to our servers.
- **Terms acceptance flag** - one-time flag in local storage.
- **Favorite teams** - locally, optional.

ProbWin AI DOES NOT COLLECT:

- Name, email, or any other personally identifiable information.
- Location data.
- Advertising identifiers (IDFA / GAID).
- User betting history (app does not accept bets).
- Payment data.

## 3. Data Processed by Our API

The app connects to our server to fetch:

- Upcoming football matches list
- Bookmaker odds (publicly available)
- Team statistics (publicly available)
- AI-generated commentaries

Our servers log **standard technical logs** (IP address, User-Agent, request timestamp) solely for operational and security purposes. Logs are deleted after 30 days.

**We DO NOT link** logs with individual users and do not create behavioral profiles.

## 4. Third-party Services

The app uses these external services:

- **the-odds-api.com** - bookmaker odds. We do NOT pass any user data to them.
- **Clubelo.com** - club Elo ratings. We do NOT pass any user data to them.
- **football-data.co.uk** - historical matches. We do NOT pass any user data to them.
- **Anthropic (Claude API)** - AI commentary generation. Requests contain ONLY public match data (team names, statistics). They contain NO user data.
