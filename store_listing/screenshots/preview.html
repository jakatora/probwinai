Jak skonwertowac SVG na PNG (3 sposoby)

Sposob A: Chrome DevTools (najlatwiej, lokalnie)
  1. Otworz pojedynczy plik SVG bezposrednio w Chrome (np. drag-drop pliku 01_welcome.svg do zakladki)
  2. Wcisnij F12 -> DevTools
  3. Wcisnij Ctrl+Shift+P -> wpisz "Capture full size screenshot" -> Enter
  4. PNG zapisze sie w Downloads w rozdzielczosci 1290x2796
Sposob B: CloudConvert (online, najwygodniejszy)
  1. Wejdz na cloudconvert.com/svg-to-png
  2. Wgraj wszystkie 6 plikow SVG na raz
  3. Ustaw Output Resolution: 1290x2796
  4. Convert -> pobierz ZIP z PNG-ami
Sposob C: Inkscape (local, batch)

W PowerShell, w folderze store_listing/screenshots:

foreach ($svg in Get-ChildItem *.svg) {
  inkscape $svg.FullName --export-type=png --export-width=1290 --export-height=2796
}
